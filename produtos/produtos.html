Conheça um pouco mais sobre o CellCode

O CellCode é um aplicativo GRATUITO, voltado para pessoas que não tem uma máquina desktop para programar. Temos a versão Standard, onde a única coisa que você precisa fazer para ter acesso ao CellCode é cadastrar-se no site e entrar na lista de espera.

Temos também a opção para os apoiadores, que tem um alguns privilégios dentro do aplicativo:*

Apoiador Standard

Valor: R$ 10,00

Apoiador Deluxe

Valor: R$ 20,00

Apoiador Full

Valor: R$ 30,00

CellCode é um editor de código desenvolvido pela FrontEnders, destinado para o desenvolvimento web e criação de softwares de diversas linguagens como JavaScript, C/C++, Java, PHP, Python, entre outras.

É uma ferramenta muito simples porém possui uma imensa loja de extensões, que continua crescendo. O usuário pode adicionar diversas funcionalidades ao CellCode de forma bem simples.

Qualquer usuário pode criar uma extensão e publicar na loja, desse modo sempre há novas ferramentas que podem ajudar o desenvolvedor.

Com ele o usuário pode digitar e editar seus códigos, em diversas linguagens de programação. O App tem suporte em diversos sistemas operacionais, Android, IOS, Microsoft, entre outros.
